# wanderlust
Airbnb clone web project - Wanderlust

## ✨ Features
- User authentication (signup/login)
- Add and manage property listings
- Responsive design

## 🛠️ Tech Stack
- Frontend: HTML, CSS, JavaScript, EJS
- Backend: Node.js, Express
- Database: MongoDB
- Other: Passport.js / JWT
